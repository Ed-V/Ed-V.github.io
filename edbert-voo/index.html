Technology Update

The site used PHP in order to help with code reusability, but it was also wasteful as the server was just regenerating the same code every time. The solution was to use a static site generator which acts like a programming language, but generate pure html at the end. Hugo was selected from the list of candidates because it was a general purpose generator compared to the others which were more oriented to blogs.

Technology Used

Sass, Hugo, Foundation

History
Wordpress to PHP Assisted Static Website

The initial website was built with Wordpress which offered an easy way to quickly add content but it lacked in the ability to customize. Futhermore, it was overly complicated and complex for a personal website and if something went wrong it would be a headache to fix. Built with Sass, PHP, Foundation
