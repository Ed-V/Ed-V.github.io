Ask Mr.Robot Generator

Download

Project Created

This project was made because of the tediousness of copy and pasting every stat over to the web page every time you wanted the website to recaculate the stats. This utility will export stat weightings from Sim Craft into a format that can be quickly imported into ask Mr.Robot. To use this save the Sim Craft html results and open them with this utility. After opening the html file export by pressing Ctrl-S or save from the menu. Open the saved text file and copy and paste it in to Ask.Mr Robot.

This project is no longer being maintained and is working as of SimCraft 6.1.2

Ask Mr.Robot no longer supports importing of stats and this project is kept for archival purposes
